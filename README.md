# ✋🎨 AirDraw AI – Gesture Controlled Drawing App

An AI-powered drawing application that lets you draw in the air using hand gestures in real-time!

Built using Python, OpenCV, and MediaPipe, this project uses computer vision to track hand movements and convert them into digital drawings.

---

## 🚀 Features

- ✋ Hand detection using webcam
- ☝️ Draw using index finger
- ✌️ Erase using gesture
- 🎨 Real-time smooth drawing canvas
- 📷 Live camera integration
- ⌨️ Press 'Q' to exit

---

## 🛠️ Tech Stack

- Python
- OpenCV
- MediaPipe
- NumPy
